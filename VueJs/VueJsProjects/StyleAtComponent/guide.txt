2.  컴포넌트의 스타일은 <style> 태그 내에서 작성한다.
    하지만 이것은 전역 스타일이므로 페이지 전체에서 사용되고
    다른 컴포넌트에서도 동일한 CSS 클래스명을 사용한다면 충돌이 발생한다.
    특정 컴포넌트만의 스타일을 지정하려면 
    1) 범위 CSS (Scoped CSS)
    2) CSS 모듈(CSS Module)
    의 두 가지 방법을 사용할 수 있다.

    이 프로젝트에서는 범위 CSS를 다룬다.
